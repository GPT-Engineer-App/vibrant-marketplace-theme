# vibrant-marketplace-theme

A WordPress theme similar to the Envato Market place, featuring a marketplace layout, vibrant color schemes, modern fonts, and a responsive design for various devices. The theme should include pages for product listings, detailed product pages, a search function, and user profile management.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/vibrant-marketplace-theme.git
cd vibrant-marketplace-theme
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
